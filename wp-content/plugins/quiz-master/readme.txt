== Description ==
This adds a quiz function to your website.  Use the [quizmaster quiz=1] shortcode on post or page to display your quiz.  Able to handle Multiple Choice and True/False questions. Capable of storing a library of questions and randomly selecting a certain amount.  Features email sending to admin or any other email address upon person completing quiz.  Also gives option to show score, incorrect and correct answers to the person taking it upon completion.

== Installation ==

= Minimal Requirements =
* WordPress: 2.8 or newer

= Recommended Requirements =
* WordPress: 2.9.1 or newer

= Basic Installation =
* Upload plugin dir to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Set plugin parameters for your needs
